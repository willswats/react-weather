## Setup

1. Run `npm install`.
2. Create a `.env.local` file.
3. Generate an API key from [openweathermap.org](https://openweathermap.org).
4. Insert the API key into the `.env.local` file, with the key as `REACT_APP_WEATHER_API_KEY`.
